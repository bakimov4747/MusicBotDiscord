# MusicWave 🎧

Discord музыкальный бот для проигрывания музыки в голосовых каналах.

## Возможности
- 🔍 Поиск и проигрывание треков с YouTube (по названию или ссылке)
- 📋 Очередь треков
- ⏯️ Пауза / Продолжить
- ⏭️ Пропустить трек
- 🛑 Остановить и выйти из канала
- 🔊 Регулировка громкости
- 📺 Информация о текущем треке

## Команды
| Команда | Описание |
|---------|----------|
| `/play <запрос/ссылка>` | Найти и воспроизвести трек |
| `/skip` | Пропустить текущий трек |
| `/pause` | Поставить на паузу |
| `/resume` | Продолжить воспроизведение |
| `/stop` | Остановить и выйти из канала |
| `/queue` | Показать очередь треков |
| `/nowplaying` | Показать текущий трек |
| `/volume <0-100>` | Установить громкость |
| `/ping` | Проверка работы бота |

## Установка и запуск

1. **Клонируй репозиторий:**
   ```bash
   git clone https://github.com/your-username/musicwave-bot.git
   cd musicwave-bot
   ```

2. **Установи зависимости:**
   ```bash
   npm install
   ```

3. **Настройка:**
   Открой `src/config.js` и вставь токен бота:
   ```js
   export default {
     token: 'ТВОЙ_ТОКЕН_БОТА',
     // ...
   };
   ```

4. **Установи FFmpeg:**
   - **Windows:** Скачай с https://ffmpeg.org/ и добавь в PATH
   - **Linux:** `sudo apt install ffmpeg`
   - **macOS:** `brew install ffmpeg`

5. **Запусти:**
   ```bash
   npm start
   ```

## Требования
- Node.js 20+
- FFmpeg
- Discord Bot Token ([создать здесь](https://discord.com/developers/applications))
